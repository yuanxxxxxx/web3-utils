# 开发规范

## 组件规范

### 1. 组件命名
- 文件夹采用 **PascalCase**: `CModal/`, `ThemeToggle/`
- 组件文件固定为 `index.tsx`
- 样式文件固定为 `style.ts`

### 2. 组件结构
```
ComponentName/
├── index.tsx    # 组件逻辑
├── style.ts     # Styled-Components 样式
└── README.md    # 组件文档（可选）
```

### 3. 组件模板

```tsx
'use client' // 客户端组件需声明

import { ComponentContainer } from './style'

interface Props {
  children?: React.ReactNode
  className?: string
}

export default function ComponentName({ 
  children, 
  className 
}: Props) {
  return (
    <ComponentContainer className={className}>
      {children}
    </ComponentContainer>
  )
}
```

## 样式规范

### Styled-Components 写法

```tsx
// style.ts
import styled from 'styled-components'

export const Container = styled.div`
  display: flex;
  padding: 16px;
  background-color: ${props => props.theme.colors.background};
`

// 支持 props 类型
export const Button = styled.button<{ $active?: boolean }>`
  opacity: ${props => props.$active ? 1 : 0.5};
`
```

### 主题使用

```tsx
// 访问主题变量
background: ${props => props.theme.colors.primary};
color: ${props => props.theme.mode === 'dark' ? '#fff' : '#000'};
```

## 代码规范

### 1. Hooks 调用顺序
```tsx
// ✅ 正确：Hooks 在顶层
const { account } = useActiveWeb3React()
const [state, setState] = useState()
useEffect(() => {}, [])

// ❌ 错误：条件调用
if (condition) {
  const value = useHook() // 禁止
}
```

### 2. TypeScript 类型
```tsx
// ✅ 显式类型定义
interface User {
  address: string
  balance: bigint
}

const user: User = { address: '0x...', balance: 0n }

// ✅ 泛型使用
const data = useReadContract<bigint>({ ... })
```

### 3. 异步处理
```tsx
// ✅ async/await + try/catch
const handleClick = async () => {
  try {
    const result = await someAsyncFunction()
    toast.success('Success')
  } catch (error) {
    console.error(error)
    toast.error('Failed')
  }
}
```

### 4. 状态管理
```tsx
// ✅ Redux 使用
import { useAppDispatch, useAppSelector } from '@/store/hooks'

const dispatch = useAppDispatch()
const user = useAppSelector(state => state.user)

dispatch(setUserInfo({ ... }))
```

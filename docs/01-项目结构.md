# 项目结构

## 核心目录

```
new-cli/
├── app/                    # Next.js App Router 页面
├── components/             # 组件库
│   ├── Basic/             # 基础 UI 组件
│   └── AppSettingsPage/   # 业务组件
├── hooks/                 # 自定义 Hooks
├── lib/                   # 核心配置
│   ├── web3/             # Web3 配置
│   ├── theme/            # 主题配置
│   ├── locales/          # 国际化文件
│   └── i18n.ts           # 国际化初始化
├── store/                 # Redux 状态管理
├── request/               # API 请求封装
├── types/                 # TypeScript 类型定义
├── utils/                 # 工具函数
└── public/               # 静态资源

```

## 关键文件说明

| 文件/目录 | 用途 |
|---------|------|
| `app/` | Next.js 14+ App Router 页面路由 |
| `lib/web3/config.ts` | RainbowKit + Wagmi 配置 |
| `lib/web3/Web3Provider.tsx` | Web3 全局 Provider |
| `types/constant.ts` | 全局配置常量（链、API地址等） |
| `hooks/useActiveWeb3React.ts` | Web3 核心 Hook（地址、链ID、登出） |
| `hooks/useLoginEvmWallet.ts` | SIWE 签名登录 Hook |
| `request/index.ts` | Axios 封装（请求拦截、错误处理） |
| `store/` | Redux Toolkit 状态管理 |
| `utils/format.ts` | 格式化工具（地址、金额） |

## 技术栈

- **框架**: Next.js 16 + React 19 + TypeScript
- **样式**: Styled-Components
- **Web3**: RainbowKit + Wagmi + Viem + Ethers.js
- **状态管理**: Redux Toolkit
- **国际化**: i18next + react-i18next
- **请求**: Axios
- **UI库**: 自研组件 + rc-pagination + rc-tooltip

## 🚀 快速开始

### 安装依赖

```bash
yarn install
```

### 运行开发服务器

```bash
yarn dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本

```bash
yarn build
```

### 启动生产服务器

```bash
yarn start
```
